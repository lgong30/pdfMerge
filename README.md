PDF Merger
==========

Required Packages
-----------------

* PyPDF2


Install Dependencies
--------------------

`pip install -r requirements.txt`


Usage
-----

```shell
git clone https://github.com/lgong30/pdfMerge.git
cd pdfMerge
python ./pdfMerge.py
```
